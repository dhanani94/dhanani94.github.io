Login

By entering your Georgia Tech credentials, the application uses the CAS login system to securely log you into your T-Square profile.

Navigation

With just a tap you can easily navigate though the applicaiton. The user interface is very simplistic and makes switching between classess or screens very simple.

Announcements

Annoucements are displayed in a simple list for the specified class. This screen also gives a brief preview of the first two lines of the annoucement.

Assignments

This feature allows you to see all of your assignments for a specified class. The assingments give a general description on when the due date is, the returned grade, and if the assignment was submitted or not. Next iterations of the application will allow users to submit assignments directly though their devices.

Gradebook

The grade book provides students with a list of their assignments. The assignments that are graded show the final grade to the left. Students can click the the asignment to view instructor comments.

Architecture

T2 is purely a client to the T-Square API. No data is persisted. The UI for each task is implemented as its own Android Activity. All data is retrieved form the server using a singleton client object implementing the T-Square API. Dependencies such as the client are injected into UI components using Dagger. Events are routed to components asynchronously via another singleton event bus (using Otto). The resulting architecture keeps each component loosely coupled to its dependencies.
